# AI Mock Interview Platform

Full-stack production starter with Next.js, Prisma, NextAuth, AI routes, and Express backend.

## Run

1. `npm install`
2. Copy `.env.example` to `.env` and set `DATABASE_URL`, `NEXTAUTH_SECRET`, etc.
3. `npx prisma migrate dev` (creates database tables — required before signup/login)
4. `npm run dev`

Open http://localhost:3000
